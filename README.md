## 👨‍💻 About me
 
Fullstack developer with **7+ years of experience** building production-grade applications. I specialize in **Java / Spring Boot** on the backend and **Angular** on the frontend, with a strong focus on **microservices architecture**, event-driven systems, and clean code.
 
Currently based in **Madrid 🇪🇸**, working on **LibryX** — a modern library management platform that serves as both a portfolio project and a technical playground for cutting-edge patterns.
 
- 🔭 Building **[LibryX](https://github.com/JuanjoOrtiz/springboot-api-libryx)** · passwordless auth (OTP + JWT), Angular 20 Signals, EDA with Kafka
- 🧠 Exploring **Angular 20** · Zoneless CD, Signals, Smart/Presentational, Lazy Loading
- ⚙️ Passionate about **microservices**, **clean architecture**, and **observability stacks**
- 💡 Always aiming for **enterprise-grade** implementations, not minimal examples
